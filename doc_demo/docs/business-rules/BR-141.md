## Rule Statement

> When a record from the import file has an unknown or invalid record type, write the record to an error file.

## Condition

The record type in the import file is not recognized as a valid type (Customer, Account, Cross-Reference, Transaction, or Card).

## Action

Write the invalid record to an error file for further investigation.
