## Rule Statement

> When processing a customer record, populate the customer's core banking system record with the data from the external export file.

## Condition

A customer record is being processed.

## Action

The customer's name, address, contact details, and other relevant information are transferred from the export file to the corresponding fields in the core banking system's customer table.
