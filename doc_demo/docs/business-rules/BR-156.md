## Rule Statement

> If a record from the external file is of an unrecognized type, it should be flagged as an error.

## Condition

The record type in the external file does not match any of the expected types (Customer, Account, Cross-Reference, Transaction, or Card).

## Action

Write the invalid record to an error file for further investigation and correction.
