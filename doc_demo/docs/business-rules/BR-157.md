## Rule Statement

> Only transactions of type 'Credit', 'Debit', or 'Adjustment' are processed for inclusion in the customer statement.

## Condition

When a transaction record is read.

## Action

If the transaction type is not 'Credit', 'Debit', or 'Adjustment', the transaction is ignored and not included in the statement.
