## Rule Statement

> Only transactions of type 'P', 'D', 'W', or 'I' are included in the customer's monthly statement.

## Condition

When processing a transaction record.

## Action

If the transaction type is not 'P', 'D', 'W', or 'I', the transaction is skipped and not included in the statement.
