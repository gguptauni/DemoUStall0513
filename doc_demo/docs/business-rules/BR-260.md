## Rule Statement

> If a transaction is successfully processed, the corresponding transaction category balance is updated to reflect the transaction amount.

## Condition

A transaction has been successfully validated and posted.

## Action

The transaction amount is added to the appropriate transaction category balance in the TCATBAL file.
