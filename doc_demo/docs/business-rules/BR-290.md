# Business Rule: Report Date Range Validation

| Paragraph | `0500-DATEPARM-OPEN` |

## Rule Statement

> The report will only be generated if the specified start date is before the specified end date.

## Condition

The program is initializing the report parameters and processing date ranges.

## Action

If the start date is not before the end date, the report generation process will likely be terminated or an error will be raised.
