## Rule Statement

> The system accepts a transaction report request from the user.

## Condition

When the user submits the transaction report request via the CICS terminal.

## Action

The system populates header information, submits a job to the internal reader for background processing, writes job submission details to a temporary data queue, and sends a confirmation screen back to the user or returns to the previous screen.
