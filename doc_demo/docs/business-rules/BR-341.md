# Business Rule: Transaction Display Limit Reached

| Paragraph | `MAIN-PARA` |

## Rule Statement

> If the maximum number of transactions that can be displayed on a single screen has been reached, stop adding more transactions to the display.

## Condition

The number of transactions to display equals the maximum allowed on the screen.

## Action

Halt the process of adding transactions to the current screen display.
