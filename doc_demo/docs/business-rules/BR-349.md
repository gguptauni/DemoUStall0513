# Business Rule: Prevent Paging Beyond First Page

| Attribute | Value |
|-----------|-------|
| Rule ID | `BR-349` |
| Category | WORKFLOW |
| Program | [COTRN00C](../programs/COTRN00C.md) |
| Paragraph | `PROCESS-PAGE-BACKWARD` |
| Lines | 1319 - 1358 |
## Rule Statement

> The user cannot navigate to a previous page if they are already on the first page of transaction data.

## Condition

The user attempts to page backward when the current display is already showing the first page of transactions.

## Action

The program will not display a previous page. The display will remain on the first page.
