## Rule Statement

> When attempting to read the previous transaction, if the beginning of the transaction file is reached, indicate that there are no more previous transactions to display.

## Condition

The program attempts to read the previous transaction from the transaction file, but the beginning of the file has been reached.

## Action

The program sets a flag or indicator to signal that no more previous transactions are available for display.
