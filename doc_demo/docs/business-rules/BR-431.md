> The new password must meet complexity requirements before it can be accepted.

## Condition

The user attempts to update their password.

## Action

The system validates the new password against predefined complexity rules (e.g., minimum length, character types). If the password does not meet the requirements, an error message is displayed, and the password update is rejected.
