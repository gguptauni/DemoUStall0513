## Rule Statement

> The system must verify that the user attempting to delete an account has the necessary permissions to perform this action.

## Condition

Before deleting a user account.

## Action

The system checks the user's security profile to ensure they are authorized to delete user accounts. If not authorized, the deletion process is halted.
